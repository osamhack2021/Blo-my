## Hyperledger Explorer 구현
준비(needs a running hyperledger network in place)
```bash
$ cd explorer
$ npm install
$ node prepare.js
```
실행
```bash
$ docker-compose up -d
```
종료
```bash
$ $ docker-compose down -v
```
[localhost:8080](localhost:8080)
